Company Profile

Shijie99 was established in February 2014.
It is an Internet technology company that has obtained China's high-tech qualifications.
It focuses on building procurement and distribution systems for the airline ticket industry through technical means.
The company helps partners to quickly integrate the resources of airline ticket suppliers from all over the world and build B2B / B2C online sales platforms based on three major GDSs (Sabre, Amadeus, Travelport)

系统核心

依托于优秀的技术架构设计，系统覆盖了机票从采购-定价-销售-收单-出票-退改-财务的全流程功能

GDS access function

GDS access function, which can be achieved through parameter configuration to access the API data interface of Sabre, Amadeus, Travelport（Worldspan/Galileo）, LCC, as long as there is an account, online sales can be completed in 20 minutes.

OTA sales channel

The OTA sales channel can be connected to the API of Skyscanner, Momondo, Wego, Kayak, Ctrip, Qunar, Alitrip through the parameter configuration

OWN sales channel

With its own sales channels, it provides different booking methods, such as B2C retail website, B2B same trade website, API interface, etc. for customer sources, and supports sales in different languages and currencies.

Ticket support

It supports automatic / manual tickets issuance mechanism, also supports setting different ticket issuance strategies.

统一中后台

Realize a unified background for ticket order processing, including functions such as ticket issuance, voiding, rescheduling, refund, exchange, and automatic cancellation of PNR.

Policy Database

Air ticket policy database system, to achieve the maintenance and management of different rebate policies for various global suppliers, the system automatically calculates and selects the optimal offer.

定价策略

The price increase / decrease /comparing/releasing system compares the price advantages and disadvantages of different suppliers and flexibly sets the pricing strategy for sales.

Config

Whitelist configuration, scheduled launch, block airlines and routes.

Exchange rate

Exchange rate management, flexible setting of exchange rates between multiple currencies。

Interface

Interface mechanism, which can connect with OA, ERP, CRM, FMS, B2B, B2C, etc. on the level of interface.
